## 📖 About the Project

PrepWise AI is a full-stack web application developed as our **Final Year Major Project**. It is designed to simplify placement preparation by bringing multiple resources together on one platform.

The platform enables students to:
- 📄 Analyze resumes using AI
- 🎤 Practice mock interviews
- 💻 Solve DSA problems
- 🧠 Improve aptitude skills
- 🏢 Prepare for company-specific interview questions
- 📈 Track preparation progress

Our objective was to create a single platform that assists students throughout their placement journey while providing an intuitive and modern user experience.
